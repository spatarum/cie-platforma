Import PNA

Importă fie fișierul sursă PNA, fie template-ul complet pentru import masiv al proiectelor și al actelor UE.
Descarcă template import Înapoi la PNA
Format recomandat

Template complet Excel

Folosește template-ul pentru importul tuturor câmpurilor structurate ale proiectelor PNA.

  • Proiecte_PNA — un rând = un proiect
  • Acte_UE — un rând = un act UE atașat unui proiect
  • Liste — valori de referință pentru câmpurile cu selecție
Descarcă template
Compatibilitate

Fișierul original PNA

Importul acceptă și fișierul folosit deja pentru monitorizarea PNA (Acțiuni_PNA). În acest mod sunt preluate câmpurile de bază, instituțiile și actele UE disponibile în sursă.

Reguli import
  • Update după: cod unic → nr. acțiune + scop → denumire + scop.
  • Instituțiile noi sunt create automat dacă nu există în listă.
  • Actele UE existente se păstrează; importul doar adaugă / actualizează legăturile din fișier.
Ce se poate importa din template
denumire, descriere, capitol / foaie de parcurs, status implementare, instituții, contacte, termene, complexitate, prioritate, expertiză internă, volum de muncă, expertiză externă, parteneri, costuri, riscuri, rapoarte de extindere, planul de creștere economică, necesitatea avizării Comisiei Europene, indicatori PNA, note, executor, costuri agregate și acte UE relevante.
