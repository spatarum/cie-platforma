Reguli:
  • Creare: dacă id este gol, se creează un chestionar nou.
  • Actualizare: dacă id este completat și există, se actualizează chestionarul.
  • Întrebările se pot actualiza doar dacă nu există răspunsuri; altfel rămân neschimbate.
  • Termen limită: folosește formatul 2026-02-15 23:59 sau 15.02.2026 23:59.
  • General: coloana este_general = da/nu. Dacă este General, capitolele/criteriile sunt ignorate.
  • Capitole: numere separate prin ; (ex: 23;24). Foi de parcurs: FID;RAP;SD.
  • Întrebări: completează intrebare_1 ... intrebare_20 (minim una).
  • Pentru chestionarele nou create, sistemul trimite notificări email către experții relevanți.
Cum pregătești fișierul
  1. Descarcă șablonul CSV de mai sus.
  2. Deschide-l în Excel și completează rândurile.
  3. Salvează ca CSV UTF-8 (recomandat).
  4. Încarcă fișierul aici și apasă Importă.
Notificări
Pentru chestionarele noi, platforma va trimite automat email către experții relevanți (General → toți experții activi).
